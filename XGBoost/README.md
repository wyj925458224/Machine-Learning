#### <span id = "1">1.什么是XGBoost</sapn>
XGBoost是经过优化的分布式梯度提升库，旨在高效，灵活且可移植。它在Gradient Boosting框架下实现了机器学习算法。XGBoost提供了并行树增强（也称为GBDT，GBM），可以快速准确地解决许多数据科学问题。相同的代码在主要的分布式环境（Hadoop，SGE，MPI）上运行，并且可以解决数十亿个示例以外的问题。
